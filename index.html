Configurar Supabase

Crea la tabla contratos en Supabase con el siguiente SQL:
Ver SQL → 

create table contratos (
  id uuid primary key default gen_random_uuid(),
  contratante_nombre text not null,
  contratante_tel text,
  contratante_dir text,
  fecha_evento date not null,
  lugar text not null,
  ciudad_firma text,
  duracion_horas numeric default 2,
  monto_total numeric not null,
  anticipo numeric not null,
  saldo numeric not null,
  fecha_limite_saldo date,
  incluye_sonido boolean default false,
  medio_pago text,
  status text default 'pendiente',
  firma_base64 text,
  firmado_at timestamptz,
  created_at timestamptz default now()
);
alter table contratos enable row level security;
create policy "read" on contratos for select using (true);
create policy "insert" on contratos for insert with check (true);
create policy "update" on contratos for update using (true);

¿Borrar contrato?

Esta acción es permanente. El contrato y la firma se eliminarán de la base de datos.
